Button Configuration

**Current Configuration**
Controller 1:
    - Left Joystick: Y axis for drive forward/reverse, X axis for rotation
    - Left Trigger: Lift motor reverse
    - Right Trigger: Lift motor forwards
Controller 2:
    - Right Joystick: Y axis for boom angle motor, X axis for boom rotate motor
    - Left Trigger: Boom extension motor reverse
    - Right Trigger: Boom extension motor forwards
    - A Button: Right clamp servo release
    - B Button: Right clamp servo close
    - X Button: Left clamp servo release
    - Y Button: Left clamp servo close

** PLANNED Competition Configuration **
Controller 1 (Driver):
    - Left Joystick: X axis for wheel Rotation
    - Right Joystick: X axis for Rotation
    - Left Trigger: Drive reverse
    - Right Trigger: Drive forward
Controller 2 (Manipulator):
    - Left Joystick: Y axis for lift motor
    - Right Joystick: X axis to control boom rotation, Y axis to control boom angle
    - Left Trigger: Boom retract
    - Right Trigger: Boom extend
    - A Button: Right clamp servo release
    - B Button: Right clamp servo close
    - X Button: Left clamp servo release
    - Y Button: Left clamp servo close
